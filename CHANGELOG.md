# Changelog

All notable changes to SEO Forge will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-04-15

### Added
- 10-phase autonomous SEO blog pipeline (CONFIG → TREND → KEYWORD → OUTLINE → DRAFT → SCORE → OPTIMIZE → EDIT → RESCORE → PUBLISH)
- 6 AI agent definitions: Keyword Hunter, Content Architect, Quality Scorer, SEO Optimizer, Competitor Spy, Publisher
- 8 rotating blog templates: Deep Reviewer, Tutorial Expert, Industry Analyst, Problem Solver, Beginner's Guide, Storyteller, Deep Comparison, Case Study
- 4-axis quality scoring system (SEO Quality, E-E-A-T Compliance, Content Depth, Reference Authority) with 0-100 scale
- Keyword grading tiers: S+, S, A+, A, B, C with SERP analysis scoring formula
- Google E-E-A-T compliance framework
- CLI tool (`seo_forge.py`) with 7 subcommands: init, trend, score-keyword, article, score-article, report, state
- Multi-platform deployment support: Next.js, Astro, Hugo, Gatsby, WordPress, Ghost, Framer
- Blog configuration JSON Schema template
- Trusted reference domain whitelist
- Competitor analysis framework
- Deployment guide for GitHub + Vercel
- Convergence loop with max 3 optimization iterations
